# Intelligent Skill Gap & Competency Analyzer 🎓

## 📌 Project Overview
The **Intelligent Skill Gap & Competency Analyzer** is a real-time smart application developed using Python and Streamlit. It is designed specifically for students at **CHRIST (Deemed to be University)** to bridge the gap between their current academic proficiency and industry-standard expectations for various high-demand roles in computer science.

This project was developed as part of the **MCA519-3 Advanced Python Programming** course (Component 3 Assignment).

## 🚀 Key Features
- **Industry Benchmarking**: Mapped competency levels for core roles: AI Engineer, Data Scientist, Full Stack Developer, and Cyber Security Analyst.
- **Expanded Industry Link Hub**: Provides a comprehensive set of **5+ links** (certifications, documentation, and courses) for every skill category to help bridge competency gaps.
- **Real-time Gap Analysis**: Automatically calculates specific areas where improvement is needed.
- **Readiness Meter**: A visual progress indicator showing overall career readiness percentage.
- **AI-Powered Project Ideator**: Dynamically generates project blueprints (Beginner to Advanced) specifically designed to bridge your most critical skill gap.
- **Personalized Roadmap**: Provides specific **Christ University MCA Course Codes** and industry certification suggestions based on identified gaps.
- **Optimized Performance**: Clean, refactored Python codebase for high-speed responsiveness.

## 🛠 Tech Stack
- **Python**: Core logic and data processing.
- **Streamlit**: Web interface and dashboard framework.
- **Plotly**: Advanced data visualizations and interactive charts.
- **Pandas/NumPy**: Data handling and mathematical computations.

## 📂 Project Structure
- `app.py`: The primary application file containing the UI and logic.
- `task.md`: Development roadmap and progress tracker.
- `walkthrough.md`: Detailed feature and setup guide.

## ⚙️ How to Run
1. **Clone the repository** (or navigate to the folder).
2. **Install dependencies**:
   ```bash
   pip install streamlit plotly pandas numpy
   ```
3. **Run the app**:
   ```bash
   python3 -m streamlit run app.py
   ```
